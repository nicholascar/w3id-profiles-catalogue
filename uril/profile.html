URI List Profile

This profile is a data model that is for lists of URIs. It is entirely described by the Media Type https://w3id.org/mediatype/text/uri-list.

The reason for assigning this URI, https://w3id.org/profile/uri-list to this profile, rather than just re-using the Media Type URI, https://w3id.org/mediatype/text/uril, is so that descriptions, like this, can be made that help with the use of the profile as a data spacification, not just as a format.

In addition to human-readable descriptions of this profile, this text, there is also a machine-readable description of this profile available which uses the Profiles Vocabulary to describe this profile in Semantic Web terms and which relates it various parts together (guidance documents like this, specification docs, format checkers etc.).

This machine-readable version is available by using Content Negotiation against this URI, https://w3id.org/profile/uril, to get information according to the Media Type text/turtle, for example, this cURL command:

  • curl --header 'Accept: text/turtle' https://w3id.org/profile/uril

For convenience, this machine-readable version is also available via this direct link:

  • https://w3id.org/profile/uril.ttl

As of November, 2019, this machine-readable version, and this document you are reading now, are not complete but they hopefully will by by the end of 2019.
