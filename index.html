Profiles Catalogue

A profiles catalogue is being established here, https://w3id.org/profile/, that will list profiles characterised according to the W3C's Profiles Vocabulary.

This is a general-purpose catalogue – it's for use by anyone – and will allow profiles to be both identified with a persistent URI (htt ps://w3id.org/profile/) and for their content to be indexed, as long as it's presented according to the Profiles Vocabulary. Profil es with URIs may be linked to by other profiles ("profiles of profiles") or be used as conformance targets allowing instance data to indicate that it claims conformance to that profile.

This catalogue is being built by the editors of the Profiles Vocabulary so see that document for their contact information.
