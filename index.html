Profiles Catalogue

This is a catalogue of profiles, that is, instances of the W3C's Profiles Vocabulary's prof:Profile class.

This is a general-purpose catalogue – it's for use by anyone – and allows profiles to be both identified with a persistent URI (https://w3id.org/profile/<YOUR-ID>) and for their content to be indexed, as long as it's presented according to the Profiles Vocabulary.

Access

This Catalogue can be accessed by humans at this web address, or by machines which may use HTTP Content Negotiation.

The various profiles and Media Types by which this Catalogue may be accessed are listed and described by its Alternate Profiles profile.

Additions

Additions to this catalogue will be accepted as long as they meet these conditions

  1. contains a primary profile description document in RDF, conforming to the Profiles Vocabulary specification
    • see the Profiles Vocabulary's Test Suite to check conformance
  2. contains an HTML description document, reflecting the RDF primary profile description document's contents, for human reading
  3. any other artifacts referred to by the profile that are not otherwise available via public URI

See the existing profiles for examples. You might view them in this Catalogue's code repository.

To add to this catalogue, please either:

  • email Nicholas Car at nicholas.car@surroundaustralia.com
  • create a Pull Request for the repository which manages the catalogue content which is https://github.com/nicholascar/w3id-profiles-catalogue

The Pull Request method is preferred!
