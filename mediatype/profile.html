Media Types Profile

This will soon be a profile, as defined by the [Profiles Vocabulary](https://www.w3.org/TR/dx-prof/), of [Dublin Core Terms](https://dublincore.org/specifications/dublin-core/dcmi-terms/) used to describe [Media Types](https://en.wikipedia.org/wiki/Media_type) in [RDF](https://www.w3.org/RDF/).
